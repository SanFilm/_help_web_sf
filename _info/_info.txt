## Требования к вёрстке:
----------------------------------------------------------------------------------
Центрирование строчного тега по горизонтали и по вертикали:
----------------------------------------------------------------------------------

HTML:
  <header class="header">
    <a class="header__logo" href="#"><img src="..." alt="LOGO"></a>
  </header>
----------------------------------------------------------------------------------

    --- 1. ---
CSS:

.header {
  text-align: center;  /* center по горизонту */
  height: 100px;
}
.header__logo{
  line-height: 100px;  /* center по вертикали (размер родителя = дочернему) */
  padding-right: 40px;

      --- 2. ---
CSS:

.header {
  display: flex;
}
.header__logo{
  margin: auto;         /* Располагает по центру блока */
  margin-right: 40px;   /* Отступ справа */

----------------------------------------------------------------------------------
